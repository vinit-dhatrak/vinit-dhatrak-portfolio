Lesson 1: The Simplest Neuron

Objectives

  • Understand the absolute basic components of a neuron: inputs, weights, and a bias.
  • Learn how a neuron combines these components to produce an output (the "forward pass").
  • Relate the function of a simple neuron to the familiar algebraic equation y = mx + c.
  • Write a complete C++ program to model a single neuron and make a prediction.

Theory

Welcome to the heart of neural networks! At its core, a single neuron (or "perceptron") is just a simple computational unit. It's a concept you're likely already familiar with from basic algebra.

Remember the equation for a straight line? y = mx + c. In neural network terminology, x is the input, m becomes the weight, c becomes the bias, and y is our output.

The formula is transformed into: output = (weight * input) + bias. This process of taking inputs and calculating an output is called the Forward Pass.

Code Example

This program will predict the price of a pizza based on its size. Our simple "model" has a base price (the bias) and a price per inch (the weight).

#include <iostream>
#include <vector>

// Lesson 1: The Simplest Neuron - A Manual Linear Regression Model

int main() {
    // --- 1. Define the Neuron's Parameters (Weight and Bias) ---
    // These are the "learned" values. For this lesson, we set them manually.
    double weight = 1.5; // The "m" in y = mx + c. Represents price per inch.
    double bias = 5.0;   // The "c" in y = mx + c. Represents the base price.

    // --- 2. Provide Input Data ---
    std::vector<double> pizza_sizes = {8, 10, 12, 14, 16};

    std::cout << "--- Pizza Price Predictor ---" << std::endl;
    
    // --- 3. The Forward Pass (The Core Logic) ---
    for (double size : pizza_sizes) {
        // The core formula: output = (weight * input) + bias
        double predicted_price = (weight * size) + bias;
        std::cout << "  - Size: " << size << "\" -> Predicted Price: $" << predicted_price << std::endl;
    }

    return 0;
}

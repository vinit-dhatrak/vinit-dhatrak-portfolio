Lesson 2: Measuring Error - The Loss Function

Objectives

  • Understand why we need a way to measure a model's performance.
  • Learn the concept of a loss function.
  • Implement the Mean Squared Error (MSE) for regression problems.

Theory

A loss function compares the model's predictions to the actual, correct values (ground truth) and outputs a single number representing the model's error. The entire goal of training is to minimize the value of the loss function.

For regression, we use Mean Squared Error (MSE). The formula is: MSE = (1/n) * sum((Actual - Predicted)^2). We square the error to make it positive and to penalize larger errors more heavily.

Code Example

We'll now compare our pizza predictor's output to the actual prices and calculate the MSE.

#include <iostream>
#include <vector>
#include <cmath>

int main() {
    double weight = 1.5;
    double bias = 5.0;

    std::vector<double> pizza_sizes = {8, 10, 12, 14, 16};
    std::vector<double> actual_prices = {17, 21, 23, 25, 30};

    std::vector<double> predicted_prices;
    for (double size : pizza_sizes) {
        predicted_prices.push_back((weight * size) + bias);
    }

    double total_squared_error = 0.0;
    for (size_t i = 0; i < actual_prices.size(); ++i) {
        double error = actual_prices[i] - predicted_prices[i];
        total_squared_error += std::pow(error, 2);
    }

    double mean_squared_error = total_squared_error / actual_prices.size();
    std::cout << "Mean Squared Error (Loss): " << mean_squared_error << std::endl;

    return 0;
}
